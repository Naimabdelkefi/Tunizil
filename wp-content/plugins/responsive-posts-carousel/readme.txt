Displays Posts, WooCommerce Products and Custom Post Types as Carousel using Shortcodes

== Description ==
Responsive Posts Carousel lets you to displays Posts, WooCommerce Products or any other Custom Post Type as Carousel. You can use from a bunch of pre made styles for post thumbnails and featured images. You can also create carousel for just post titles without having thumbnails. Custom Post Meta can also be used to display in carousel. Select the posts by post type or taxonomy, term and specific posts can also be exclude from list.<br>

== Installation ==
1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Responsive Posts Carousel' and install it
3. Go to Dashboard > Carousel > Add New
4. Setup settings and copy shortcode
5. Paste that shortcode in any page and visit that page
